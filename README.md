# PSVSCodium
Module to Manage Portable VSCodium Install

## Invoke-VSCodiumUpdate
- Sample
```PowerShell
Invoke-VSCodiumUpdate -VSCodiumEXEPath "C:\Temp\VSCodium\VSCodium.exe" -VSCodiumTempPath "C:\Temp\"
```
